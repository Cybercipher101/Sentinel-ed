# Audio Assets Directory

Please place your ElevenLabs generated audio file in this directory and rename it to `alert.mp3`.

1. Generate your warning voice in ElevenLabs (e.g., "AI detected a potential violation. Please look at your screen.").
2. Download the `.mp3` file.
3. Move the downloaded file into this `public/audio` directory.
4. Rename it to exactly `alert.mp3`.

The React application is pre-programmed to automatically play `/audio/alert.mp3`.
